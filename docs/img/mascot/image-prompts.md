# Zyme — AI Image Generation Prompts

Self-contained prompts for generating all 7 pose variants of Zyme the Yeast Cell.
Each prompt includes the full base character description so prompts can be used
independently. Generate at 512×512 or 1024×1024 pixels with a transparent background,
then save to `docs/img/mascot/` with the filename shown.

---

## Base Character Description (embedded in every prompt below)

> A flat cartoon vector illustration of Zyme, a friendly golden-yellow yeast cell
> (Saccharomyces cerevisiae) mascot for a Food Science textbook. Zyme is a small,
> round, bubbly cell body colored golden-yellow (#f9a825), with tiny circular lab
> safety goggles perched near the top of the cell, large bright expressive eyes,
> and small CO2 bubbles floating nearby. Clean flat-vector style with green (#2e7d32)
> and orange (#f57c00) accent details. Transparent background. No text in the image.

## After Generating Images

1. Save all 7 PNGs to `docs/img/mascot/` with the exact filenames shown above.
2. Run the padding trimmer on each image to remove excess transparent space:

```bash
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/neutral.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/welcome.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/thinking.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/tip.png

python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/celebration.png
```
